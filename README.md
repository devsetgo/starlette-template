# starlette-template
Simple Starlette template to start off an app with.

### Goals
- [ ] Basic layout pattern for a Starlette App
- [ ] User login
 - [ ] Athentication
 - [ ] Authorization
- [ ] UI Template to work from
    - [ ] Example Pages
- [ ] Health Status (basic functions like Spring Actuator)
- [ ] Prometheus (Metrics) endpoint
