This project is built to be used as a template for Starlette based sites. It is free to use as you wish. I would like to know who, where or what it is used for. Feel free to email me at mike ryan 56 @ gmail . com.

Contributions to improve, fix, or add functions are welcome!

This template is built with Starlette, AdminLTE 3.0 HTML template, and Uvicorn as the ASGI Server. There is a Docker Image available to try.

  • Starlette.io
  • AdminLTE.io
  • Uvicorn ASGI Server
  • Docker Image

                  ### Use it
                  - git clone https://github.com/devsetgo/starlette-dashboard
                  - create a virtualenv on linux or Windows Subsystem Linux
              
                  #### Run APP
                  First Change directory to app

                  UVICORN
                  Development:
                  ~~~~
                  uvicorn main:app --port 5000 --reload
                  python3 main.py
                  ~~~~
                  Production:
                  ~~~~
                  uvicorn main:app --workers 2
                  gunicorn -c gunicorn_cfg.py main:app
                  # Note: gunicorn is the config for the dockerfile
                  ~~~~

                  Docker
                  ~~~~
                  Docker: docker pull mikeryan56/starlette-dashboard:latest (not implemented)
                  ~~~~

                  #### Run Tests
                  ~~~~
                  python3 -m pytest
                  ~~~~

                  #### Create coverage badge
                  ~~~~
                  coverage-badge -o coverage.svg -f
                  ~~~~

                  #### Pre-Commit & Hooks
                  ~~~~
                  - Follow install instructionsL: [https://pre-commit.com/#install](https://pre-commit.com/#install)
                  - pre-commit install
                  - pre-commit run -a
                  ~~~~
